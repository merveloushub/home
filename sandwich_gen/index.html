Cursed Sandwich Generator

Information

There are 24 different types of buns, ingredients, and condiments— A total of items available.

Generation

  • There can be 0-3 condiments generated.
  • There can be 1-5 ingredients generated.
  • Buns will match 25% of the time.

Rationale

  • Items had to be things that are typically put in a sandwiches, OR is a key item in an established sandwich.
  • If an item appeared on Wikipedia's List of Sandwiches more than once, then I added the ingredient to the generator.
  • This is why there are some weird items on the list.
  • If there is an item generated that makes you go "why the hell is that there?", that's probably why.
  • Here's an explanations for some of the following sandwich ingredients:
    • Kiwis, oranges, strawberries, and whipped cream: from Japanese sandos.
    • Salsa: a common component in Latin American sandwiches, such as choripáns.
    • Sprinkles: from fairy bread.
  • If a sandwich ingredient is not specified (e.g., chutney, pâté, fried fish) then use your imagination.
